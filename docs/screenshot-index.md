## 管理端

当前 README 未收录管理端截图，但项目已包含管理端路由与页面：仪表盘、用户管理、图书管理、评论管理、作品审核、稿件审核、章节审核和 RBAC 管理。后续补截图时建议按“治理概览、内容审核、权限配置”三组补充。

## 项目边界

截图中的内容、用户、阅读数据和创作者数据均为演示数据。项目展示的是全栈业务原型能力，不代表真实版权引入、真实收益结算或真实平台运营。
